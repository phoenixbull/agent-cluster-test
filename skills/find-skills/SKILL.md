# Find Skills

Search and discover available skills for OpenClaw.

## Usage

Use when:
- User asks what skills are available
- Need to find a skill for a specific task
- Exploring new capabilities

## Commands

```bash
# List all available skills
openclaw skills list

# Search ClawHub
npx clawhub search <query>

# Get skill info
openclaw skills info <skill-name>
```

## Examples

- "What skills do I have for search?"
- "Find skills for working with GitHub"
- "Show me all available skills"
