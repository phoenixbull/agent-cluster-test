# Vector Memory

Semantic memory storage using vector databases.

## Features

- Semantic search
- Long-term memory
- Context retrieval
- Similarity matching
- Knowledge graphs

## Usage

Use for:
- Storing conversation history
- Semantic search across memories
- Context-aware responses
- Knowledge retrieval
- Pattern recognition

## Supported Backends

- LanceDB
- Pinecone
- Weaviate
- Chroma
- Qdrant

## Examples

- "Search my memories for project discussions"
- "Find similar conversations"
- "Store this in long-term memory"
