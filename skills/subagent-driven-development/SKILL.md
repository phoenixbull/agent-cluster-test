# Subagent-Driven Development

Orchestrate multiple specialized agents for complex tasks.

## Features

- Task decomposition
- Agent coordination
- Parallel execution
- Result aggregation
- Quality control

## Usage

Use for:
- Large development projects
- Multi-step workflows
- Parallel task execution
- Complex problem solving

## Agent Types

- **Coder** - Code generation and refactoring
- **Reviewer** - Code review and quality checks
- **Tester** - Test creation and execution
- **Designer** - UI/UX design
- **Architect** - System design and planning

## Workflow

```
1. Decompose task → 2. Assign agents → 3. Execute in parallel
                    ↓
4. Aggregate results ← 5. Quality check ← 6. Deliver
```

## Examples

- "Build a complete web application"
- "Refactor this codebase"
- "Create tests for all modules"
