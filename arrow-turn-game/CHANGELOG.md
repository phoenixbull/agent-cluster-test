### 🗺️ 关卡设计

基于市场调研和关卡设计原理：

#### 难度曲线
```
第 1-3 关   → 教学关（简单障碍，熟悉操作）
第 4-8 关   → 入门关（Z 字形障碍，练习拐弯）
第 9-15 关  → 进阶级（迷宫障碍，需要规划）
第 16-20 关 → 挑战级（复杂障碍，考验技巧）
```

#### 关卡类型
1. **直线型** - 基础教学
2. **Z 字形** - 练习连续拐弯
3. **螺旋型** - 空间感知
4. **迷宫型** - 路线规划
5. **动态障碍** - 时机把握（后续版本）

### 🎵 音频系统

使用 Web Audio API 合成音效：

| 音效 | 场景 | 频率 |
|------|------|------|
| **转弯** | 90 度转向 | 800Hz 短音 |
| **得分** | 收集检查点 | 1200Hz 清脆音 |
| **碰撞** | 撞到障碍/边界 | 200Hz 锯齿波 |
| **升级** | 关卡完成 | 600-1200Hz 滑音 |

### 📊 游戏数值

#### 难度公式
```javascript
难度系数 = (箭头数量 × 0.3) + 
           (拐弯次数 × 0.2) + 
           (路线长度 × 0.1) + 
           (障碍密度 × 0.4)

推荐范围:
- 1-3 关：0.5 - 1.0
- 4-8 关：1.0 - 2.0
- 9-15 关：2.0 - 3.5
- 16-20 关：3.5 - 5.0
```

#### 速度递进
```
第 1 关：150 像素/秒
每关增加：10 像素/秒
第 20 关：340 像素/秒
```

#### 分数计算
```
基础分数：每移动 1 像素 = 1 分
检查点：+100 分/个
连击奖励：连击数 × 10 分
关卡奖励：+200 分/关
```
