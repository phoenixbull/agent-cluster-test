## ⚠️ 注意事项

1. **HTTP 不是 HTTPS** - 当前是 HTTP 协议，浏览器可能提示不安全（正常）
2. **端口可能被防火墙阻止** - 需要配置安全组
3. **服务器重启后需要重新启动** - 可以配置 systemd 服务
4. **生产环境建议** - 使用 Nginx 反向代理 + HTTPS

---

## 🔒 生产环境配置（可选）

### 使用 Nginx 反向代理

```nginx
server {
    listen 80;
    server_name game.yourdomain.com;
    
    location / {
        proxy_pass http://127.0.0.1:9000;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
    }
}
```

### 配置 HTTPS

```bash
# 使用 Let's Encrypt
certbot --nginx -d game.yourdomain.com
```

## 📞 问题排查

### 无法访问公网 IP

1. 检查安全组是否开放 9000 端口
2. 检查服务器防火墙
3. 确认游戏服务器在运行

### 游戏加载失败

1. 检查浏览器控制台错误
2. 确认文件权限正确
3. 清除浏览器缓存

### 移动端无法操作

1. 确保使用触摸事件（已支持）
2. 检查浏览器兼容性
3. 尝试横屏/竖屏切换
