### 添加 PRD 完成通知

```python
# notifiers/dingtalk.py
def notify_prd_complete(self, workflow: Dict, prd_info: Dict):
    """Phase 1: PRD 完成通知"""
    title = f"📄 PRD 完成 - {workflow.get('id', 'N/A')}"
    
    text = f"""## 📄 PRD 文档完成

**工作流**: {workflow.get('id', 'N/A')}
**产品经理**: {prd_info.get('pm_name', 'N/A')}
**需求**: {prd_info.get('requirement', 'N/A')[:50]}

---

### 📋 产出物

- PRD 文档：{prd_info.get('prd_url', 'N/A')}
- 用户故事：{prd_info.get('user_stories', 0)} 个
- 验收标准：{prd_info.get('acceptance_criteria', 0)} 个

---

🤖 Agent 集群自动通知
"""
    return self.dingtalk.send_markdown(self.admin_user_ids, title, text)
```

# notifiers/dingtalk.py
def notify_critical_bug(self, bug: Dict):
    """Phase 4: 严重 Bug 通知"""
    title = f"🐛 严重 Bug - {bug.get('id', 'N/A')}"
    
    text = f"""## 🐛 发现严重 Bug

**Bug ID**: {bug.get('id', 'N/A')}
**严重程度**: 🔴 {bug.get('severity', 'critical')}
**模块**: {bug.get('module', 'N/A')}

---

### 📋 Bug 详情

**标题**: {bug.get('title', 'N/A')}

**描述**: {bug.get('description', 'N/A')[:100]}

**复现步骤**: {bug.get('reproduction_steps', 'N/A')}

---

⚠️ 需要优先处理
"""
