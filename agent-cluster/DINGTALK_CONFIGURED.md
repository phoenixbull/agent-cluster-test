## 🚀 使用方式

### 在代码中使用

```python
from notifiers.dingtalk import ClusterNotifier

# 从配置加载
notifier = ClusterNotifier(
    webhook_url="https://oapi.dingtalk.com/robot/send?access_token=...",
    secret="SEC..."
)

# 发送 PR 就绪通知
notifier.notify_pr_ready(task, result)

# 发送任务失败通知
notifier.notify_task_failed(task, result, failure_reason)
```

### 自动触发

通知已集成到以下流程：
- ✅ Ralph Loop - 失败时自动通知
- ✅ Monitor - 监控到 PR 就绪时通知
- ✅ Cluster Manager - 任务完成时通知
