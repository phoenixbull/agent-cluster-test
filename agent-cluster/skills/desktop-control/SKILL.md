# Desktop Control

Automate desktop applications and system UI.

## Features

- Window management
- Application control
- Keyboard/mouse automation
- Screen capture
- Clipboard operations

## Usage

Use for:
- Automating desktop tasks
- Controlling applications
- UI testing
- Screen recording
- Batch operations

## Requirements

- Platform-specific tools (AppleScript, AutoHotkey, etc.)
- Accessibility permissions
- Screen recording permissions (macOS)

## Examples

- "Open Safari and go to example.com"
- "Take a screenshot of the current window"
- "Type 'Hello World' in the active window"
