# Browser Automation

Automate web browser interactions using Playwright.

## Features

- Web page automation
- Form filling and submission
- Screenshot capture
- Web scraping
- End-to-end testing

## Usage

Use for:
- Automating web tasks
- Testing web applications
- Extracting web data
- Taking screenshots
- Web form automation

## Commands

```bash
# Open browser
openclaw browser open <url>

# Take screenshot
openclaw browser screenshot

# Execute JavaScript
openclaw browser eval "<js-code>"
```

- "Open example.com and take a screenshot"
- "Fill out the contact form"
