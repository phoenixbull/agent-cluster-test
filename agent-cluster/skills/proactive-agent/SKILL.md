# Proactive Agent

Make the agent proactive with automatic checks and scheduled tasks.

## Features

- Heartbeat monitoring
- Scheduled task execution
- Automatic status checks
- Proactive notifications

## Usage

Use for:
- Setting up periodic checks (email, calendar, etc.)
- Creating automated workflows
- Scheduling reminders and tasks
- Monitoring systems proactively

## Configuration

Edit `HEARTBEAT.md` to add periodic tasks:

```markdown
# Daily checks
- Check email for urgent messages
- Review calendar for upcoming events
- Monitor system status
```

## Examples

- "Check my calendar for today"
- "Set up daily status checks"
- "Monitor my inbox every hour"
