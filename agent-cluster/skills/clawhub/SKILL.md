# ClawHub CLI

Package manager for OpenClaw skills.

## Features

- Search skills
- Install skills
- Update skills
- Publish skills
- Version management

## Installation

```bash
npm install -g clawhub
```

## Usage

```bash
# Search for skills
npx clawhub search <query>

# Install skill
npx clawhub install <skill-name>

# Update skills
npx clawhub update

# Publish skill
npx clawhub publish <skill-path>

# List installed skills
npx clawhub list
```
