## 📋 项目需求

我需要一个个人任务管理系统，可以创建和管理待办事项，支持分类、优先级、截止日期，界面简洁现代，支持移动端访问。

**核心功能**:
1. 用户注册登录
2. 任务 CRUD（创建、查看、编辑、删除）
3. 分类管理
4. 优先级设置（高/中/低）
5. 截止日期提醒
6. 完成状态标记
7. 搜索筛选
8. 统计面板（完成率、待办数量）
9. 响应式设计（手机 + 电脑）
