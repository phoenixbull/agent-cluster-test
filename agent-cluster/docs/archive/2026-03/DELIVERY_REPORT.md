## 🎯 游戏特性

### 物理参数
| 参数 | 值 | 说明 |
|------|-----|------|
| 蓄力时间 | 0-2000ms | 按住时间决定跳跃距离 |
| 跳跃力度 | 10-25 | 水平速度 |
| 重力 | 0.5 | 每帧重力加速度 |
| 方块宽度 | 120px | 标准方块 |
| 方块间距 | 100-200px | 随机生成 |

### 分数规则
| 行为 | 分数 |
|------|------|
| 成功落在方块上 | +1 |
| 落在方块中心（完美） | +2 |
| 最高分记录 | 本地存储 |

### 难度曲线
| 分数段 | 变化 |
|--------|------|
| 0-10 | 标准间距 100-200px |
| 11-20 | 最大间距 250px |
| 21-30 | 最大间距 300px，出现冰面 |
| 30+ | 最小间距 150px |

## 🚀 如何运行

### 方式 1: 直接打开
```bash
# 在浏览器中打开
open index.html
```

### 方式 2: 使用 LayaAir CLI
```bash
# 安装 LayaAir CLI
npm install -g laya-cli

# 运行项目
laya run

# 构建发布版本
laya build

# 构建微信小游戏版本
laya build --platform wx
```

### 方式 3: 本地服务器
```bash
# 使用 Python
python3 -m http.server 8080

# 访问 http://localhost:8080
```
