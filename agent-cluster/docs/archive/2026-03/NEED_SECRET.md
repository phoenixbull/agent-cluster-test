# 🔑 钉钉加签密钥配置

## ❌ 当前问题

钉钉机器人开启了**加签保护**，需要提供加签密钥才能发送消息。

错误信息：
```
errcode: 310000
errmsg: 机器人发送签名不匹配
```

---

## ✅ 解决方案（二选一）

### 方案 1: 提供加签密钥（推荐）

#### 第 1 步：获取加签密钥

1. 打开钉钉群
2. 群设置 → 智能群助手
3. 找到 **"Agent 集群助手"** 机器人
4. 点击机器人 → 设置
5. 查看 **加签密钥** (SEC 开头的字符串，例如：`SECxxxxxxxxxx`)

#### 第 2 步：提供密钥

将加签密钥发给我，我会帮你完成测试和配置。

格式：`SEC` 开头的字符串

---

### 方案 2: 临时关闭加签保护（快速测试）

#### 步骤：

1. 打开钉钉群 → 群设置 → 智能群助手
2. 找到 **"Agent 集群助手"** 机器人
3. 点击编辑
4. 安全设置 → 选择 **"自定义关键词"**
5. 添加关键词：`通知`、`PR`、`任务`
6. 保存

这样就不需要加签密钥了，但安全性会降低。
