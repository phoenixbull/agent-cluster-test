## 4. 智能重试机制流程

```
任务执行失败
        │
        ▼
┌─────────────────────────────────┐
│ 1. 失败原因分类 (15+ 种)         │
│                                 │
│ ┌─────────────────────────────┐│
│ │ Prompt 相关 (3 种)            ││
│ │ • PROMPT_AMBIGUOUS          ││
│ │ • PROMPT_INCOMPLETE         ││
│ │ • PROMPT_CONTRADICTORY      ││
│ └─────────────────────────────┘│
│ ┌─────────────────────────────┐│
│ │ 模型相关 (3 种)              ││
│ │ • MODEL_HALLUCINATION       ││
│ │ • MODEL_TIMEOUT             ││
│ │ • MODEL_OUTPUT_INVALID      ││
```
