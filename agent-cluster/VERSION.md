## 📋 版本规范

### 版本号格式

```
vX.Y.Z
│ │ │
│ │ └─ 修订版本号 (Bug 修复、小优化，向后兼容)
│ └─── 次版本号 (新功能、向后兼容)
└───── 主版本号 (破坏性变更、架构调整)
```

### 发布规则

| 变更类型 | 版本升级 | 示例 |
|---------|---------|------|
| Bug 修复、小优化 | Z +1 | v2.1.0 → v2.1.1 |
| 新功能 (向后兼容) | Y +1, Z=0 | v2.1.3 → v2.2.0 |
| 破坏性变更 | X +1, Y=0, Z=0 | v2.2.0 → v3.0.0 |

## 🔄 版本升级流程

### 升级前准备

1. **备份当前版本**
   ```bash
   cd /home/admin/.openclaw/workspace/agent-cluster
   tar -czf backup_v$(date +%Y%m%d_%H%M%S).tar.gz .
   ```

2. **查看当前版本**
   ```bash
   cat VERSION.md | grep "当前版本"
   ```

### 升级步骤

1. **更新版本号**
   - 编辑 `VERSION.md`
   - 更新"当前版本"字段
   - 添加新版本历史记录

2. **更新配置文件**
   ```json
   {
     "cluster": {
       "version": "2.2.0"
     }
   }
   ```

3. **更新文档**
   - README.md
   - AI_PRODUCT_DEV_AGENT_SUMMARY.md

4. **提交变更**
   ```bash
   git add .
   git commit -m "chore: 升级到 v2.2.0 - 智能增强版"
   git tag v2.2.0
   git push origin main --tags
   ```

### 回滚方案

```bash
# 解压备份
tar -xzf backup_v20260319_130210.tar.gz

# 恢复特定文件
cp backup_*/monitor.py .
```
