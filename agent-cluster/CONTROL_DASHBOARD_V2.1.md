## 📝 快速命令

### 启动集群

```bash
cd /home/admin/.openclaw/workspace/agent-cluster
python3 orchestrator.py "项目需求描述"
```

### 启动监听器

```bash
bash start_deploy_listener.sh
```

### 查看状态

```bash
ps aux | grep -E "orchestrator|deploy_listener"
tail -f /tmp/deploy_listener.log
```

### 停止监听器

```bash
kill $(ps aux | grep deploy_listener | grep -v grep | awk '{print $2}')
```
